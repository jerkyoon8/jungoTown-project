# SpringCommunity

## 프로젝트 소개
Spring Boot를 기반으로 한 커뮤니티 웹 애플리케이션입니다. 사용자 관리, 게시글 작성, 댓글, 채팅, 상품 관리 등의 기능을 제공합니다.

## 기술 스택
- **Language**: Java
- **Framework**: Spring Boot
- **Template Engine**: Thymeleaf
- **Database**: MySQL
- **ORM**: MyBatis
- **Library**: Lombok
- **Cache**: Redis

## 주요 기능
- 회원 가입 및 로그인 (OAuth 2.0 예정)
- 상품 등록 및 조회
- 실시간 채팅 (WebSocket)
- 공지사항 및 알림

## 실행 방법
```bash
./gradlew bootRun
```
